# 우파루 크로스 확률 검색기

## 개요
우파루 오딧세이의 크로스 확률표를 편리하게 확인하고 사용자가 원하는 추가 확률을 확인하기 위한 확률 검색 페이지

데이터 참고:
> https://wooparoo-odyssey.hangame.com/probability <br>
  계정레벨 30, 우파루레벨 20, 행운업 O, X

## 라이브러리

- Selenium
  - 웹페이지 작업 자동화
- BeautifulSoup
  - HTML, XML 데이터 추출
- Pandas
  - CSV 데이터 분석 및 조작
- PapaParse
  - CSV 파이릉ㄹ 파싱하여 JSON 데이터로 변환
- React
  - UI 구성 및 앱 개발
